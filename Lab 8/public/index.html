Preguntas Lab 2 - HTML

1. Diferencia entre Internet y Web

Internet es la red global que conecta computadoras. La Web es un servicio que funciona sobre Internet usando HTTP.

2. Partes de un URL

Esquema, dominio, ruta, parámetros y fragmento.

3. Métodos HTTP

GET obtiene información, POST envía datos, PUT reemplaza, PATCH actualiza parcialmente, DELETE elimina.

4. Método para enviar usuario y contraseña

Se usa POST porque no envía los datos en la barra de direcciones.

5. Método al acceder a una página

Normalmente GET.

6. Código 200

Significa que la petición fue exitosa.

7. Error 404

Es recurso no encontrado. El desarrollador debe revisarlo si es error del sitio.

8. Error 500

Es un error interno del servidor. Generalmente lo debe corregir el desarrollador.

9. Deprecated

Significa que ya no se recomienda usarlo. Ejemplo: font, center, frame.

10. Diferencias HTML4 y HTML5

HTML5 agrega etiquetas semánticas, mejor multimedia y mejores formularios.

11. Componentes de tabla

table, tr, th, td.

12. Controles de formulario

input, select, textarea, button.

Preguntas Lab 3 - CSS

1. Uso de !important

No es recomendable usarlo mucho porque dificulta el mantenimiento del código.

2. Imagen de fondo

Debe elegirse bien porque puede afectar la visibilidad y el rendimiento.

4. CSS minificado

Preguntas Lab 5 - Frameworks

1. Describe Material Design

Material Design es un sistema de diseño creado por Google que busca crear interfaces modernas, limpias y consistentes. Se basa en el uso de sombras, profundidad, animaciones suaves y colores bien definidos para simular superficies físicas en el entorno digital. Su objetivo es mejorar la experiencia del usuario haciendo que las interfaces sean intuitivas, visualmente atractivas y fáciles de usar en diferentes dispositivos.
