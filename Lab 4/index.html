1) ¿Diferencias y semejanzas entre Java y JavaScript?

                    Se parecen en sintaxis básica (llaves, if/for) y ambos soportan POO. Java corre en la JVM y suele ser tipado estático.
                    JavaScript es tipado dinámico y se ejecuta en navegador o Node.js.
                  


                  
2) Métodos del objeto Date (5+):

                    getFullYear(), getMonth(), getDate(), getDay(),
                    getHours(), getMinutes().
                  


                  
3) Métodos de arreglos (5+):

                    push(), pop(), shift(), unshift(), slice(),
                    splice(), map(), filter().
                  


                  
4) Variable local dentro de una función:

                    Con let o const dentro de la función/bloque.
                  


                  
5) Implicaciones de usar variables globales:

                    Puede causar efectos secundarios, errores difíciles de depurar y conflictos porque cualquier parte del código las modifica.
